# 基金筛选系统 — 设计文档

**日期：** 2026-06-16
**状态：** 已确认

---

## 1. 项目概述

构建一个 A 股公募基金的智能筛选系统。核心能力：
- 多维度综合评分，对全市场基金排名
- 基本面筛选 + 技术面择时的混合策略
- 区分「一笔买入」和「定投」两种场景
- 完整的策略回测验证

## 2. 市场与标的选择

| 维度 | 选择 |
|------|------|
| 市场 | A 股公募基金（全类型） |
| 基金类型 | 股票型、混合型、指数型、债券型、货币型 |
| 重点 | 偏股型 + 指数型 |
| 分类方式 | 按基金类型分别排名，不跨类比较 |

## 3. 综合评分模型

### 3.1 一笔买入模式 — 五维度打分

**总分公式：**
> 总分 = 收益×30% + 估值×15% + 风控×12% + 基本面×13% + 技术×30%

| 维度 | 权重 | 指标 | 指标权重 |
|------|------|------|----------|
| 📈 收益 | 30% | 近1月/3月/6月/1年/3年收益率 | 5/8/8/6/3 |
| 💰 估值 | 15% | PE分位(指数)/净值偏离度(主动)/PB分位/回撤判断 | 6/4/3/2 |
| 🛡️ 风控 | 12% | 最大回撤/夏普比率/波动率/Calmar比率 | 6/3/2/1 |
| 🏛️ 基本面 | 13% | 规模/成立年限/经理年限/费率/经理业绩 | 3/3/3/2/2 |
| 📊 技术 | 30% | 均线趋势/MACD/RSI/布林带 | 12/8/5/5 |

### 3.2 定投模式 — 不同逻辑的五维度

**总分公式（定投）：**
> 总分 = 估值×25% + 长期趋势×20% + 收益×20% + 波动率×15% + 风控×10% + 基本面×10%

核心差异：估值权重最高（起点不能贵）、波动率单独评估（靠波动摊薄成本）、长期趋势必须向上。

### 3.3 标准化方法

- 每个指标先在**同类型基金**中做**百分位标准化**（0~100）
- 反向指标（回撤、波动率、费率）用反向百分位
- 区间最优指标（规模、RSI、回撤）用分段函数
- 缺失数据的指标赋予中性分 50

## 4. 数据层

### 4.1 数据源
- **AkShare**（主）：基金列表、净值、指数数据
- **东方财富接口**（备）：基金详情、规模、费率、经理

### 4.2 更新频率

| 数据 | 频率 |
|------|------|
| 净值数据 | 每日盘后 18:00 |
| 基金列表 | 每周 |
| 基金详情 | 每周 |
| 指数数据 | 每日 |
| 评分计算 | 每日（净值更新后自动触发） |

### 4.3 数据库表（SQLite）

- **fund_basic** — 基金基本信息（代码、名称、类型、规模、经理、费率等）
- **fund_nav** — 每日净值时序（代码、日期、单位净值、累计净值、日收益）
- **fund_score** — 评分结果（代码、日期、总分、各维度分、同类排名）
- **fund_signal** — 技术指标（MA/MACD/RSI/布林带）
- **backtest_record** — 回测记录

## 5. 回测引擎

### 5.1 回测逻辑
1. 设定回测区间 + 调仓周期
2. 每个调仓日用**当时已知数据**重新评分，选 Top-N
3. 等权买入，持有到下一个调仓日
4. 计算策略组合收益 vs 基准（沪深300/中证500）

### 5.2 输出指标
累计收益、年化收益、最大回撤、夏普比率、胜率、超额收益(Alpha)、信息比率

### 5.3 定投回测
单独支持定投模拟：每月固定金额投入，按当时价格买入，累加份额，计算 IRR

## 6. Web 页面

4 个页面，极简实用：

1. **基金筛选**（首页）— 筛选条件 + Top-N 排名表 + 模式切换（一笔/定投）
2. **基金详情** — 基本信息 + 评分雷达图 + 净值走势 + 技术信号
3. **回测报告** — 策略收益曲线 vs 基准 + 核心指标 + 历年对比
4. **策略配置** — 权重调整 + 参数设置

## 7. 技术栈

| 层 | 技术 |
|------|------|
| 后端 | Python 3.11+ / FastAPI |
| 前端 | Vue 3 + ECharts |

| 任务调度 | APScheduler（定时拉数据+算分）|

## 9. 约束与注意事项

- AkShare 免费接口有频率限制，采集需加随机延迟
- 首次全量拉取耗时 30~60 分钟，后续增量 5~10 分钟
- 评分按类别分别排名，不跨类比较
- 回测防止过拟合：分训练期/验证期
- 所有权重和参数可自定义
