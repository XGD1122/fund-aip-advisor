# 指数基金定投决策系统 — 选基逻辑说明文档

> 版本: 2.1 | 日期: 2026-06-17
> 
> **v2.1 重大新增：** 定投时机顾问引擎 —— 买入/卖出信号检测、定投额度建议、止盈策略、持仓诊断
> 
> **v2.0 重大变更：** 聚焦指数型基金、新增跟踪误差维度、修复AIP评分引擎Bug、优化定投逻辑

---

## 一、总体架构

系统采用 **多因子综合评分模型**，对每只**指数型基金**计算 0~100 分的综合得分，按指数子类型分别排名，按总分从高到低输出筛选结果。

**适用范围：** 仅指数型基金（含指数型-股票、指数型-债券等子类型），不再覆盖主动管理型基金。

**两套评分模式：**

| 模式 | 场景 | 核心问题 |
|------|------|----------|
| 一笔买入 | 当前有一笔资金，想选最好的指数基金买入 | 谁现在最值得买？ |
| 定投筛选 | 每月固定金额投入，长期坚持 | 谁现在最值得开始定投？ |

## 二、一笔买入模式

### 2.1 评分公式

```
总分 = 估值分×25% + 收益分×22% + 风控分×18% + 技术分×15% + 基本面分×10% + 跟踪误差分×10%
```

**v2.0 权重调整：** 估值权重从15%→25%（指数基金买得便宜最关键），技术面从30%→15%（指数基金技术信号弱），新增跟踪误差10%（指数基金核心指标）。

### 2.2 各维度详解

#### 💰 估值维度（权重 25%）— 第一核心

**目的：判断指数基金当前是否"贵"，避免追高买入。**

| 指标 | 权重 | 计算方式 | 评分逻辑 |
|------|------|----------|----------|
| PE 分位（模拟）| 15% | 当前净值在近 2 年（504天）中的百分位 | 分位 ≤ 30% → 90~100分；30~70% → 60~80分；70~85% → 40~60分；>85% → 0~40分 **大幅扣分** |
| 净值偏离 MA120 | 6% | (当前净值 - 120日均线) / 120日均线 | -5%~+10% 健康(90分)；>20% 高位(0~30分) |
| 近期回撤状态 | 4% | (当前净值 - 60日高点) / 60日高点 | 健康回调 5~10% 加分；>20% 深跌谨慎 |

**高位风险标签（v2.0 统一为85%阈值）：**
- 净值分位 ≥ 85%：标记 **"高位⚠️"**
- 净值分位 ≥ 80%：标记 **"偏高⚡"**
- 净值分位 ≤ 20%：标记 **"低位✅"**
- 净值分位 ≤ 40%：标记 **"偏低👍"**

满足以下任一条件触发 **高位警告（high_warning）**：
- 净值分位 ≥ 85%
- 近 3 月涨幅 > 20%
- MA120 偏离 > 20%

---

#### 📈 收益维度（权重 22%）

衡量基金的历史回报能力。**v2.0 降低短期权重，长期权重占比提升**。

| 指标 | 权重 | 计算方式 | 方向 |
|------|------|----------|------|
| 近 1 月收益率 | 3% | (当前净值 - 22日前净值) / 22日前净值 | 越高越好 |
| 近 3 月收益率 | 5% | (当前净值 - 66日前净值) / 66日前净值 | 越高越好 |
| 近 6 月收益率 | 5% | (当前净值 - 132日前净值) / 132日前净值 | 越高越好 |
| 近 1 年收益率 | 6% | (当前净值 - 252日前净值) / 252日前净值 | 越高越好 |
| 近 3 年年化收益率 | 3% | (1 + 总收益)^(1/3) - 1 | 越高越好 |

短期:长期 ≈ 13:9，比 v1.0 的 21:9 更加平衡。

**归一化方法（v2.0 收紧）：** `得分 = max(0, min(100, 50 + 原始值 × 200))`
- 收益为 0% → 50 分
- 收益为 +25% → 100 分（指数基金收益区间更窄）

---

#### 🛡️ 风控维度（权重 18%）

| 指标 | 权重 | 计算方式 | 评分逻辑 |
|------|------|----------|----------|
| 最大回撤 | 8% | 近一年滚动回撤最小值 | 回撤越小越好；-5%以内→90分 |
| 夏普比率 | 4% | (年化收益 - 无风险利率) / 年化波动率 | >1→75分；>2→100分 |
| 年化波动率 | 4% | 日收益标准差 × √252 | 越低越好；<15%→高分 |
| Calmar 比率 | 2% | 年化收益 / **全区间**最大回撤绝对值（v2.0修复） | 越高越好 |

---

#### 📊 技术维度（权重 15%）— v2.0 大幅降低

指数基金净值跟随标的指数，技术信号预测能力弱于个股。权重从 30% 降至 15%。

| 指标 | 权重 | 评分逻辑 |
|------|------|----------|
| 均线趋势 (MA) | 6% | MA5 > MA20 → +25分；MA20 > MA60 → +25分 |
| MACD 信号 | 3% | DIF > DEA → +20分；柱状图为正 → +20分 |
| RSI(14) | 3% | **v2.0 修复：** 40~70最优(80~100分)；>70统一递减函数，无跳变 |
| 布林带 | 3% | 净值在中轨附近偏上最优 |

---

#### 🎯 跟踪误差维度（权重 10%）— v2.0 新增

**指数基金的核心使命是跟踪指数，跟踪误差是衡量指数基金质量的第一指标。**

| 指标 | 权重 | 评分逻辑 |
|------|------|----------|
| 年化跟踪误差 | 10% | <1%→95~100分；1~2%→80~95分；2~4%→50~80分；>4% 快速扣分 |

---

#### 🏛️ 基本面维度（权重 10%）

| 指标 | 权重 | 评分逻辑 |
|------|------|----------|
| 基金规模 | 3% | 2~50 亿最优（100分） |
| 成立年限 | 3% | ≥3 年满分（100分）；不足按比例 |
| 管理费率 | 4% | 指数基金费率越低越好（核心优势） |

## 三、定投筛选模式（v2.0 重大优化）

### 3.1 评分公式

```
总分 = 估值分×30% + 趋势分×20% + 波动率分×15% + 风控分×12% + 收益分×10% + 跟踪误差分×8% + 基本面分×5%
```

### 3.2 核心设计理念

定投的本质是用时间换空间，通过持续买入摊薄成本。因此：

1. **估值决定起点质量（30%，最高权重）：** 高位开始定投 = 长期被套
2. **趋势决定方向（20%）：** 不要在下跌趋势中定投，等趋势转好再开始
3. **波动率是朋友（15%）：** 定投靠波动摊薄成本，但波动太大=风险太大

### 3.3 定投专属指标

| 指标 | 权重 | 逻辑 |
|------|------|------|
| PE 分位 | 18% | 分位越低越好，≤20% 是极佳时机 |
| MA60 斜率 | 10% | 近 20 天 MA60 变化率。斜率为正 → 中期趋势向上 |
| MA120 斜率 | 10% | 近 20 天 MA120 变化率。斜率为正 → 长期趋势向上 |
| 波动率（AIP专用）| 15% | **v2.0修复：** 年化 15%~30% 最佳(90~100分)；<10% 没效果也要扣分 |

### 3.4 定投评级

| 评级 | 条件 | 含义 |
|------|------|------|
| 极佳定投时机 🌟 | 净值分位 ≤ 20% | PE处于历史低位，是开始定投的最佳时机 |
| 适合定投 ✅ | 净值分位 ≤ 40% | 估值偏低，可以放心开始定投 |
| 可以定投 | 净值分位 40~80% | 估值中性，定投可以但非最优 |
| 谨慎定投 ⚡ | 净值分位 ≥ 80% | 估值偏高，不建议开始新定投 |
| 暂停定投 ⚠️ | 净值分位 ≥ 85% | 处于历史高位，应暂停定投等待回调 |

## 四、定投时机推荐系统（v2.1 新增）⭐

### 4.1 设计理念

定投的成功不仅取决于选什么基金，更取决于**什么时候投多少**。系统为每只指数基金提供独立的买卖时机信号，帮助用户做出三个核心决策：

1. **投不投？** — 当前是否适合继续定投？
2. **投多少？** — 正常、加倍、减半还是暂停？
3. **何时卖？** — 什么时候止盈？

### 4.2 买入信号（4个）

| 信号 | 触发条件 | 强度 | 说明 |
|------|----------|------|------|
| 🟢 估值低位 | PE分位 ≤ 20% | ⭐⭐⭐ | 历史大底区域，加倍定投的最佳时机 |
| 🟢 超跌反弹 | 距60日高点回撤 >15% 且 RSI < 30 | ⭐⭐ | 恐慌杀跌后的修复机会 |
| 🟢 趋势转好 | MA5 > MA20 且 MACD金叉（DIF>DEA>0）| ⭐⭐ | 短期趋势确认向上 |
| 🟢 均线支撑 | 净值偏离MA120 < 3% | ⭐ | 回踩长期均线支撑位 |

### 4.3 卖出/减仓信号（4个）

| 信号 | 触发条件 | 强度 | 说明 |
|------|----------|------|------|
| 🔴 估值高位 | PE分位 ≥ 85% | ⭐⭐⭐ | 历史高位区域，应暂停定投考虑减仓 |
| 🔴 趋势转坏 | MA120斜率 < -5% 且 MA5 < MA20 | ⭐⭐⭐ | 中长期趋势破位，暂停定投 |
| 🔴 短期过热 | 近3月涨幅 > 25% | ⭐⭐ | 短期涨幅过大，减半定投等回调 |
| 🔴 超买回落 | RSI > 75 且 MACD死叉 | ⭐ | 技术面见顶信号 |

### 4.4 定投额度倍数

| 倍数 | 条件 | 含义 |
|------|------|------|
| **2.0x 加倍** | PE分位 ≤ 20% 且有 ≥2 个买入信号 | 估值底部 + 信号共振，难得的重仓时机 |
| **1.5x 增投** | PE分位 ≤ 40% | 估值偏低，适度增加投入 |
| **1.0x 正常** | PE分位 40%~70%，趋势正常 | 估值合理，按计划定投 |
| **0.5x 减半** | PE分位 ≥ 80% 或 趋势转坏 | 估值偏高或趋势不稳 |
| **0.0x 暂停** | PE分位 ≥ 85% 且高位警告触发 | 极度高估，停止定投等待回调 |

### 4.5 止盈策略

| 止盈类型 | 触发条件 | 操作 |
|----------|----------|------|
| 🏁 部分止盈 | 累计收益 > 30% | 卖出 1/3 持仓，锁定部分利润 |
| 🏁 全部止盈 | 累计收益 > 50% 且 PE分位 > 70% | 卖出全部，估值偏高+利润丰厚 |

### 4.6 时机综合评分

综合四大维度（估值35% + 趋势25% + 技术信号20% + 风险状态20%）生成 0~100 的时机评分：

- **≥80分：** 极佳时机，信号强烈看多
- **60~80分：** 适合定投
- **40~60分：** 中性，正常定投
- **20~40分：** 谨慎，减半或暂停
- **<20分：** 危险，建议暂停

### 4.7 API 端点

| 端点 | 用途 |
|------|------|
| `GET /api/aip/timing/{code}?cost_nav=1.5` | 单只基金完整时机分析 |
| `GET /api/aip/opportunities?min_score=60&top_n=20` | 全市场定投机会扫描 |
| `POST /api/aip/portfolio/check` | 持仓批量诊断 |

## 五、归一化方法汇总

每个原始指标值通过以下方式映射到 0~100 分数：

| 指标类型 | 归一化函数 | 示例 |
|----------|------------|------|
| 正向指标（收益、夏普等）| `max(0, min(100, 50 + value × 200))` | 收益 25% → 100 分 |
| 反向指标（回撤、费率）| `max(0, min(100, 100 - abs(value) × 250))` | 回撤 -10% → 75 分 |
| 分位指标（PE分位）| 分段函数（低位高分、高位惩罚）| 分位 95% → 13 分 |
| 区间最优（AIP波动率）| 分段函数（15~30%=最优）| 波动率 20% → 93 分 |
| 区间最优（跟踪误差）| 分段函数（<1%最优, >4%严重）| 跟踪误差 1.5% → 87 分 |
| 区间最优（基金规模）| 分段函数（2~50亿最优）| 规模 10 亿 → 100 分 |
| 年限类（成立年限）| `min(100, value × 100 / 3)` | 1.5 年 → 50 分 |

### 分位指标详细分段函数

| 分位区间 | 得分 | 含义 |
|----------|------|------|
| 0 ~ 30% | 90 ~ 100 | 历史低位，便宜 |
| 30 ~ 70% | 60 ~ 80 | 中等位置 |
| 70 ~ 85% | 40 ~ 60 | 偏高，开始扣分 |
| 85 ~ 100% | 0 ~ 40 | 高位，严重扣分 |

**设计意图**：分位 > 85% 后曲线陡降，确保高位基金被严重惩罚。

## 六、回测引擎

### 5.1 一笔买入回测

```
1. 设定回测区间 (如 2022-01-01 ~ 2025-12-31)
2. 设定调仓周期 (如每 3 个月)
3. 每个调仓日，仅用当时已知的历史数据评分，选 Top-N (如 10)
4. 等权买入，持有到下一个调仓日
5. 模拟换仓，计算组合净值曲线
6. 对比基准 (沪深300)
```

**v2.0 关键改进：** 回测评分使用与实盘一致的五维度加权体系，而非简化公式。回测结果能真实反映策略有效性。

**关键约束：绝不使用未来数据。** 回测日期之后的数据不会被评分函数看到。

### 5.2 定投回测

```
1. 每月首个交易日投入固定金额
2. 每次定投前，用当时数据重新筛选 Top-N 定投标的
3. 等权分配金额，按当时净值买入份额
4. 累加份额，计算最终市值和 IRR（Newton-Raphson迭代求解）
```

### 5.3 回测输出指标

| 指标 | 含义 | 参考标准 |
|------|------|----------|
| 累计收益率 | 全区间总收益 | > 基准 = 策略有效 |
| 年化收益率 | 折合每年 | > 基准年化 |
| 最大回撤 | 最大亏损幅度 | < 20% = 优秀 |
| 夏普比率 | 风险调整后收益 | > 1 合格, > 2 优秀 |
| 胜率 | 调仓周期中正收益比例 | > 55% 有效 |
| 超额收益 (Alpha) | 相对基准的超额 | > 0 = 有超额能力 |
| 信息比率 | 超额收益 / 超额收益波动率（v2.0修复） | > 0.5 合格, > 1 优秀 |
| IRR（定投）| 内部收益率（v2.0 Newton迭代） | > 无风险利率 = 有效 |

### 5.4 防止过拟合

回测分为两段：
- **训练期** (2024-01-01 前)：用于观察策略表现
- **验证期** (2024-01-01 后)：验证策略是否持续有效

## 七、重要设计决策

### 7.1 为什么只做指数型基金

1. **透明度高：** 指数基金跟踪明确指数，持仓可预测
2. **费率低：** 管理费通常 0.15%~0.5%，远低于主动基金
3. **无经理风险：** 不依赖基金经理个人能力
4. **适合定投：** 跟踪指数长期向上，定投确定性高
5. **降低复杂度：** 不需要评估经理能力、持仓风格等主观指标

### 7.2 为什么新增跟踪误差维度

跟踪误差是衡量指数基金"纯度"的核心指标。跟踪误差过大说明：
- 基金可能存在较大现金拖累
- 抽样复制不够精确
- 存在隐性主动管理

对于指数投资者，跟踪误差大的指数基金偏离了"买指数"的初衷。

### 7.3 为什么定投估值权重最高（30%）

定投最怕的是在估值高位开始。如果在 PE 分位 90% 以上开始定投，即使坚持 3 年也可能不赚钱。反之，PE 分位 20% 以下开始定投，几乎所有定投都能盈利。

### 7.4 为什么 AIP 波动率是区间最优而非越低越好

定投的数学本质是靠价格波动来摊薄平均成本。如果价格是一条直线（零波动），定投和一笔买入效果一样。适度波动（15~30%）每次买入的价格不同，低位多买份额，高位少买份额，实现成本摊薄。

波动率太低（<10%）→ 摊薄效果弱；波动率太高（>40%）→ 风险太大。15%~30% 是指数基金定投的最优区间。

### 7.5 PE/PB 分位为什么用净值模拟

国内公募基金的 PE/PB 数据不公开披露。当前用 **净值历史分位** 作为替代。对于指数型基金，如果后续接入对应指数的 PE 数据（如同花顺/东方财富的指数估值接口），可显著提升精度。

## 九、使用流程

### 定投决策（推荐首页）
1. 双击 `start.bat` 或 `python start.py`
2. 浏览器打开 `http://localhost:3000`
3. 默认进入**定投决策**页面
4. 点击"扫描机会"发现当前适合定投的基金
5. 或输入基金代码进行单只诊断
6. 输入持仓进行批量诊断（成本+份额）

### 基金筛选
1. 切换到"基金筛选"Tab
2. 选择模式（一笔买入 / 定投筛选）
3. 选择指数型子类型（或全部）
4. 点击"开始筛选"
5. 查看排名，关注 **高位⚠️** 标记和**定投评级**

### 策略调优
1. 编辑 `backend/config.py` 修改权重
2. 重启后端服务
3. 重新筛选，对比结果变化

### 回测验证
1. 打开"回测报告"页面
2. 设定区间、调仓周期、Top-N
3. 运行回测
4. 对比策略 vs 基准（沪深300），判断策略是否有效
