# 指数基金 Top20 买入推荐 — 评分逻辑说明

> 版本: 3.0 | 日期: 2026-07-07
>
> **v3.0 重大变更：** 系统简化为单一 Top20 买入推荐页面，采用 Graham-Buffett 价值投资 5 维评分体系。

---

## 一、筛选流程

```
全市场基金 → A类过滤 → 活跃过滤 → 走平线过滤 → 5维评分 → Top20
```

### 1.1 A 类过滤

只保留名称以 `A` 结尾的基金（长期持有费率最优）。

### 1.2 活跃过滤

- 最新净值日期距今 ≤ 90 天
- 净值记录 ≥ 60 条（至少 3 个月数据）
- 近 1 年收益 > 2%（历史上是赚钱的基金）

### 1.3 走平线过滤

排除净值几乎不变动的"死基金"（如债券基金 003184 走成一条水平线）：

> AVG(ABS(日收益率)) < 0.0001 → 排除

---

## 二、5 维大师规则评分

### 总分构成

```
总分 = 估值分(0-30) + 均线分(0-25) + 回撤分(0-20) + 质量分(0-15) + 技术分(0-10)
满分 100
```

---

### 维度一：估值分位（30 分）— Graham 安全边际

**核心理念：** 买得便宜是最大的安全边际。用净值在近 2-5 年的历史百分位衡量估值高低。

| 净值分位 | 得分 |
|----------|------|
| ≤ 10% | 30 |
| ≤ 20% | 27 |
| ≤ 30% | 23 |
| ≤ 40% | 18 |
| ≤ 50% | 13 |
| ≤ 60% | 8 |
| ≤ 70% | 4 |
| > 70% | 1 |

分位越低 = 估值越便宜 = 分数越高。分位 > 70% 严重扣分（只给 1 分）。

---

### 维度二：均线系统（25 分）— 金字塔抄底

**核心理念：** 价格低于长期均线时买入，越低越买。三条均线构成金字塔信号。

| 条件 | 得分 |
|------|------|
| 净值 < MA60 | +7 |
| 净值 < MA120 | +8 |
| 净值 < MA240 | +10 |

- **3 条均线全破** → 25 分（最佳抄底信号）
- **2 条均线破位** → 15-18 分
- **0 条均线破位** → 0 分（价格高于所有均线，不适合买入）

**双重确认（与估值联动）：** 估值分位 ≤ 20% 且 ≥ 2 条均线破位 → 标记 ⭐"低估值+均线支撑"。

---

### 维度三：回撤买入（20 分）— 从高点回落

**核心理念：** 好基金跌了才是买点。从近 1 年最高点回撤越大，买入价值越高。

| 回撤幅度 | 得分 |
|----------|------|
| ≥ 30% | 20 |
| ≥ 20% | 18 |
| ≥ 15% | 15 |
| ≥ 10% | 12 |
| ≥ 5% | 8 |
| > 0% | 4 |
| ≤ 0%（创新高）| 1 |

创新高时不追——扣到最低分。

---

### 维度四：质量保障（15 分）— 排除价值陷阱

| 指标 | 满分 | 评分逻辑 |
|------|------|----------|
| 成立年限 | 5 | ≥ 5 年满分；不足按比例扣减 |
| 基金规模 | 5 | 规模适中最优 |
| 夏普比率 | 5 | > 1 满分；负值 0 分 |

**价值陷阱保护：** 夏普为负 → 质量 0 分（基金不创造超额收益，跌了也白跌）。

---

### 维度五：技术信号（10 分）— RSI + MACD

| 指标 | 满分 | 评分逻辑 |
|------|------|----------|
| RSI(14) | 6 | 30-40（超卖区）→ 6 分；40-50 → 4 分；> 70 → 0 分 |
| MACD | 4 | DIF > DEA 且柱状图 > 0（金叉）→ 4 分 |

---

## 三、惩罚机制

### 自由落体惩罚

近 60 日跌幅 > 25% → **总分 × 0.3**（极端下跌，即使指标好也要谨慎）。

### 高位追涨警告

近 1 年涨幅 > 25% → 标记 **"追高风险: 近1年涨幅过大"**，但不直接扣分（涨得多的基金往往估值分已经很低）。

## 四、输出字段

| 字段 | 含义 |
|------|------|
| score | 综合评分（0-100） |
| ret_5d | 近 5 日收益率 |
| ret_20d | 近 20 日收益率 |
| ret_1y | 近 1 年收益率 |
| nav_pct_2y | 2 年净值百分位 |
| drawdown | 从 1 年高点的回撤幅度 |
| ma_below | 低于几条主要均线（0-3） |
| rsi | RSI(14) 数值 |
| consecutive_down | 连续下跌天数 |
| warning | 风险提示标签 |

## 七、设计决策

### 为什么单页面？

专注一个场景：今天买什么。不分散注意力到回测、定投计算、持仓诊断——这些功能可以做，但不是每天都要用。

### 为什么 A 类基金？

A 类基金申购费前端收取，适合长期持有（≥ 1 年）。C 类收销售服务费，持有越久越贵。

### 为什么估值不在最高的"便宜"状态给满分？

30 分是上限而非 100 分——因为一只基金既便宜又处于均线下方又回撤大的状态，才是真正的"值得买"。单一维度满分没有意义，需要多维共振。

### 为什么不直接用 PE/PB？

公募基金的 PE/PB 数据不公开。当前用净值历史分位模拟估值分位。对于指数型基金，净值跟踪指数，净值分位 ≈ 指数估值分位。
